SwiftCart – E-Commerce Website

SwiftCart is a modern and responsive e-commerce frontend website built using Vanilla JavaScript, Tailwind CSS, and DaisyUI. The project focuses on creating a clean user interface, fast performance, and a smooth shopping experience.

The website allows users to browse products, explore categories, and interact with a modern UI design inspired by contemporary e-commerce platforms.

Installation & Setup

To run this project locally:

Clone the repository

git clone https://github.com/your-username/swiftcart.git

Navigate to the project folder

cd swiftcart

Open the project

You can simply open index.html in your browser or use a live server.

Deployment

This project is deployed using Netlify.

Steps to deploy:

Push project to GitHub

Connect repository to Netlify

Deploy the site automatically
